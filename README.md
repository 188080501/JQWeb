# JQWeb

一些基于WASM的迷你项目，仅供娱乐和供WASM初学者参考

目前只在Qt6.5/Qt6.7版本中进行测试，开发环境是Windows

相比Qt的标准WASM产物，本项目做了以下修改：

* 修改了HTML引导页样式

* 增加了PWA配置

* 加了字体文件，保证中文显示正常
